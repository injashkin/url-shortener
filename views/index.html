Учебный API проект для FreeCodeCamp: Микросервис, сокращающий URL адреса

Задание:

  1. Если отправить POST запрос URL-адреса в [project_url]/api/shorturl/new то в ответе получится сокращенный URL-адрес в формате JSON.
    Например: {"original_url":"www.google.com","short_url":1}
  2. Если передать неправильный URL-адрес, который не соответствует формату http(s)://www.example.com(/more/routes), ответ в формате JSON будет содержать следующую ошибку {"error":"invalid URL"}
    Подсказка: чтобы быть уверенным, что отправленный url-адрес указывает на допустимый сайт, вы можете использовать функцию dns.lookup(host, cb) из основного модуля dns.
  3. Когда я посещаю сокращенный URL, он перенаправляет меня на мою исходную ссылку.

Создание коротких URL-адресов

пример: POST [project_url]/api/shorturl/new - https://www.google.com

Пример использования:

[this_project_url]/api/shorturl/3
